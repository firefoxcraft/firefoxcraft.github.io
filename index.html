What is HTML
A html or Hypertext Markup Language that is used to create and design a website. HTML requires Critical, Technical and Artistic skills to make an attractive website that will catch the consumers eyes. However, even though most programmers use this language, the HTML is not a programming language. It is meant to design out of something and not to program out of something. Learning HTML sytaxes doesn't make you a programmer.
